# Claude 渠道切换器 - Electron 版 v3.0

一个使用 Electron 构建的现代化 Claude API 渠道管理工具。

## ✨ 特性

- 🎨 现代化的玻璃态设计
- 🌊 流畅的动画效果
- 📱 响应式布局
- 🚀 快速渠道切换
- ⚙️ 自定义终端配置
- 💾 本地配置存储

## 🚀 快速开始

### 安装依赖
```bash
cd electron-app
npm install
```

### 启动应用
```bash
npm start
```

### 构建应用（可选）
```bash
npm run build
```

## 🔄 重新加载应用

如果你修改了代码并想查看更新：

### 方法 1：重启应用
1. 关闭当前运行的应用窗口
2. 在终端中再次运行 `npm start`

### 方法 2：开发者工具重新加载
1. 在应用窗口中按 `Ctrl+Shift+I` 打开开发者工具
2. 按 `Ctrl+R` 或 `F5` 重新加载页面

### 方法 3：修改 main.js 支持热重载（推荐）
在开发时，可以按 `Ctrl+R` 直接重新加载窗口。

## 📖 使用说明

### 渠道管理
1. 点击「新建渠道」按钮
2. 输入渠道名称、API Token 和 Base URL
3. 点击「保存」

### 切换渠道
1. 在渠道列表中找到目标渠道
2. 点击「切换」按钮
3. 当前激活的渠道会显示绿色状态指示器

### 启动 Claude
1. 在激活的渠道卡片上点击「启动」按钮
2. 会在新的终端窗口中启动 Claude

### 设置
1. 点击左侧导航栏的「设置」
2. 可以修改：
   - 配置文件路径
   - 启动终端类型
